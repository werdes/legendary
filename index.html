You will need an GuildWars2 API-Key to use this site.
In order to create such a key, you have to log in to your account at account.guildwars2.com. Head to the "API Keys" page within your account page. This is where you can add a new API key.
You have to give the following set of permissions to the key: account, inventories, characters
Add new account:
The API-Keys will be saved in a cookie on your computer. No data is transmitted to any server that does not belong to Arenanet. Keys are only used for API-Requests.
